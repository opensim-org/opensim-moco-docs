OpenSim Moco Documentation

Moco is a part of OpenSim for solving optimal control problems for musculoskeletal systems defined as OpenSim models.The package has Python, MATLAB, and XML (command-line) interfaces; the core library is written in C++.

Here are some problems you can solve with Moco:

  • Solve for the muscle activity that tracks a known motion.
  • Solve for a new motion that optimizes some cost (e.g., effort).
  • Solve for mass properties that minimize residual (unrealistic) forces.
