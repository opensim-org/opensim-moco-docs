Detailed Description

TrackingController is an abstract class from which all tracking type of controllers should be derived.

This class implements convenience methods to construct desired trajectories of desired model state values, like joint angles, from Storage (file) or user-supplied functions and provides methods for obtaining the error between model and desired state values.

Derive classes need only to implement the tracking control law based on the error signals computed by this base class.
