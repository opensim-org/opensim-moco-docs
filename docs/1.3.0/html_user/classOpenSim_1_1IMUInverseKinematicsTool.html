Detailed Description

A Study that performs an Inverse Kinematics analysis with a given model.

Inverse kinematics is the solution of internal coordinates that poses the model such that the body rotations (as measured by IMUs) affixed to the model minimize the weighted least-squares error with observations of IMU orientations in their spatial coordinates.

Author
Ajay Seth
