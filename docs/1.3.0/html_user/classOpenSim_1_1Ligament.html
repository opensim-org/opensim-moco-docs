Detailed Description

A class implementing a ligament.

◆ computePathColor()

virtual SimTK::Vec3 OpenSim::Ligament::computePathColor ( const SimTK::State &  state) const
protectedvirtual

Override this method if you would like to calculate a color for use when the Ligament's path is displayed in the visualizer.

You do not have to invoke the base class ("Super") method, just replace it completely. This method will be invoked during realizeDynamics() so the supplied state has already been realized through Stage::Velocity and you can access time, position, and velocity dependent quantities. You must not attempt to realize the passed-in state any further since we are already in the middle of realizing here. Return SimTK::Vec3(SimTK::NaN) if you want to leave the color unchanged (that's what the base class implementation does).

Parameters
[in]state
A SimTK::State already realized through Stage::Velocity. Do not attempt to realize it any further.
Returns
The desired color for the path as an RGB vector with each component ranging from 0 to 1, or NaN to indicate that the color should not be changed.
