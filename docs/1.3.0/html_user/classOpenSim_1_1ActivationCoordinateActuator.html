Detailed Description


Similar to CoordinateActuator (simply produces a generalized force) but with first-order linear activation dynamics.

This actuator has one state variable, activation, with \( \dot{a} = (x - a) / \tau \), where \( a \) is activation, \( x \) is excitation, and \( \tau \) is the activation time constant (there is no separate deactivation time constant). The statebounds_activation output is used in Moco to set default values for the activation state variable. Default Property Values 

activation_time_constant: 0.01
default_activation: 0.5
