Detailed Description

Reference values to be achieved for specified Markers that will be used via optimization and/or tracking.

Also contains a weighting that identifies the relative importance of achieving one marker's reference relative to another.

Author
Ajay Seth
Examples:
exampleMarkerTracking.cpp, and exampleMarkerTracking10DOF.m.

OpenSim::MarkersReference::MarkersReference ( const TimeSeriesTable_< SimTK::Vec3 > &  markerData,
const Set< MarkerWeight > &  markerWeightSet,
Units  units = Units(Units::Meters) 
)

Form a Reference from TimeSeriesTable and corresponding marker weights.

The marker weights are used to initialize the weightings of the markers provided by the Reference. Marker weights are associated to markers by name. If a markerWeightSet is provided, then only those markers listed in the set are tracked, otherwise all the markerData table that correspond with model markers are tracked with the default weighting. The TimeSeriesTable should contain the key 'Units', representing units of the columns, as table metadata. In absence of 'Units' metadata, columns are assumed to be of units 'meters'.
