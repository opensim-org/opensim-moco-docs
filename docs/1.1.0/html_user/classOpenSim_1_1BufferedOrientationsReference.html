Detailed Description

Subclass of OrientationsReference that handles live data by providing a DataQueue that allows clients to push data into and allows the InverseKinematicsSolver to draw data from for solving.

Ideally this would be templatized, allowing for all Reference classes to leverage it.

Author
Ayman Habib
